# 参照PDF 再作図チェック表

最終更新: 2026-07-10

このファイルは、`scan-001.pdf` と `scan-001 (1).pdf` の各A4相当ページをツールで再作図できるか確認するための管理表です。参照PDF本体はリポジトリへコミットしない。

## ページ分割ルール

- 参照PDFは、A4図面をA3に2ページ面付けしたスキャンである。
- 各A3スキャンページを90度時計回りに回転する。
- 回転後の左半分を `A`、右半分を `B` として、それぞれA4縦1ページとして扱う。
- 例: `scan-001.pdf p2A` は、`scan-001.pdf` の2枚目A3スキャンを90度時計回りに回転した後の左半分。

## 判定基準

- [x] 共通機能あり: ページ再現に必要な基本操作が実装済み。
- [ ] 記号一致確認: 該当ページに出る記号の寸法、線、向き、文字位置が参照PDFと一致する。
- [ ] ページ再作図確認: 実際に該当ページをツール上で人が再作図し、PDF出力で参照PDFとの差異を確認済み。
- [ ] 不足洗い出し: ページ固有の不足記号、表現、操作を下表へ記録済み。
