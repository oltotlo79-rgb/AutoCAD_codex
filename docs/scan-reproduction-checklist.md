# 参照PDF 再作図チェック表

最終更新: 2026-07-10

このファイルは、`scan-001.pdf` と `scan-001 (1).pdf` の各A4相当ページをツールで再作図できるか確認するための管理表です。参照PDF本体はリポジトリへコミットしない。

## ページ分割ルール

- 参照PDFは、A4図面をA3に2ページ面付けしたスキャンである。
- 各A3スキャンページを90度時計回りに回転する。
- 回転後の左半分を `A`、右半分を `B` として、それぞれA4縦1ページとして扱う。
- 例: `scan-001.pdf p2A` は、`scan-001.pdf` の2枚目A3スキャンを90度時計回りに回転した後の左半分。

## 判定基準

- [x] 共通機能あり: ページ再現に必要な基本操作が実装済み。

- [ ] ページ再作図確認: 実際に該当ページをツール上で人が再作図し、PDF出力で参照PDFとの差異を確認済み。
- [ ] 不足洗い出し: ページ固有の不足記号、表現、操作を下表へ記録済み。

## 接続点グリッド整合の網羅検査(2026-07-10)

`window.__edsTest` フック経由で全パレット種別(+デザインバリエーション)を(50,47.5)へ配置し、
接続ピンの整合を自動検査した。

- **第1ピンの配線軸一致: 37種+4バリエーション全てOK**(クリックした行/グリッドへ正確一致)。
  ピンが枠中心に無い部品(押しボタン/サーマル素子/遮断器/箱物)も配置ロジックがピン基準で補正する。
- 端子台/コネクタは先頭端子ノード基準で配置・行スナップされる(node0=クリック点を確認)。
- 端子(小円)のアンカーは中心のみ(縁アンカーだと配線が0.55mmずれるため中心へ統一。
  白抜き円が配線を隠すので見た目は縁で止まる)。
- 既知の注記: 幅が2.5mm格子の倍数でない種別(接点w8、遮断器w12等)は**反対側ピンのx**が
  格子から±0.5〜2.0mm外れるが、同一水平線上のため配線の折れ・段差は発生しない
  (幅は参照実測と一括生成ビルダーの座標に合わせて維持)。junctionは点要素のため対象外。
