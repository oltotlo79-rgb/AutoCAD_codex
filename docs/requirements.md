# 電気図面作図ツール 要件定義書

## 1. 目的

AutoCAD Electrical のように電気図面を作図できる、配布しやすい単一 HTML ファイルの作図ツールを作成する。

初期段階では、ワークスペース内の `scan-001.pdf` および `scan-001 (1).pdf` に含まれる A4 電気図面を、人がツール上で忠実に再作図できることを最優先の受入条件とする。

## 2. 配布・実行条件

- 配布形式は単一の `.html` ファイルとする。
- メール添付、GitHub からのダウンロード、ローカルファイルとしての直接起動に対応する。
- 基本動作は Chrome / Edge 系ブラウザを想定する。
- 外部サーバー、インストーラ、常時ネットワーク接続を必須にしない。

## 3. 基本方針

- 操作感は PowerPoint / diagrams.net 風とする。
- 左側パレットから部品をドラッグして配置する。
- 図面は SVG ベースで表示し、内部データは JSON として保持する。
- 保存した JSON を後から読み込み、再編集できるようにする。
- PDF 印刷を必須機能とする。
- SVG / PNG / DXF / CSV などの追加出力は段階的に実装する。
- 画面表示用グリッドは作図補助として表示し、PDF/SVG/PNG 出力には含めない。

## 4. 用紙・図枠要件

- 初期状態は A4 縦とする。
- 初期状態で枠線、内枠、座標番号、表題欄を表示する。
- A4 縦、A4 横、A3 縦、A3 横を選択できる。
- 参照PDFにある複数パターンの詳細記載欄に対応するため、図枠/表題欄テンプレートを選択可能にする。
- 表題欄の各項目は、プロジェクト情報またはページ情報からマッピングして更新できる。
- 初期テンプレートは以下を用意する。
  - 基本: 右下表題欄
  - 詳細: 右下表題欄 + 改訂欄
  - 側面表題欄
  - 目次/表紙: 大型表欄
  - 枠線のみ

## 5. ページ管理要件

- 複数ページをタブで切り替えられる。
- ページ追加、複製、削除、名称変更ができる。
- ページ順を前後へ移動できる。
- 各ページごとに用紙サイズ、向き、図枠パターン、表題欄情報を保持する。
- PDF 印刷時は全ページを印刷対象にする。
- 参照PDFのように A3 縦へ A4 相当の図面を 2 枚面付けして印刷できる。
- ラダー回路図ページをテンプレートから追加できる。
- 現在ページへラングを追加できる。
- PNG/JPEG/WebP/SVG 画像を下絵として現在ページへ挿入できる。
- 下絵画像は透明度、位置、サイズ、回転、ロックを編集でき、帳票とクロス参照には含めない。
- ロック中の下絵画像は通常のクリック/範囲選択を邪魔せず、専用ボタンから選択できる。
- 下絵画像は全体、上半分、下半分、左半分、右半分の表示範囲を選択できる。
- 下絵画像は既定でPDF/SVG/PNG出力に含めず、確認用に含めたい場合だけ設定でONにできる。
- `scan-001.pdf` と `scan-001 (1).pdf` は、A4図面をA3に2ページ面付けしたスキャンである。各A3スキャンページを90度時計回りに回転し、左半分/右半分をそれぞれA4縦1ページとして扱う。
- 2面下絵ページ作成では複数画像を選択でき、各画像を90度時計回りに回転したうえで左/右A4ページを連続作成できる。

## 6. 作図要件

初期版で以下の図形・記号を作図できること。

- 文字
- 直交配線
- 汎用直線
- 連動破線
- 矩形
- 円
- 信号矢印
- 接続点
- 端子
- 端子台
- 端子配線セット
- ラダー/ラング
- a 接点
- b 接点
- 接点ブロック
- コイル
- ランプ
- ブザー
- 機器箱
- 設備枠
- 場所枠
- PLC / I-O ブロック
- コネクタ
- 表
- ヒューズ
- 遮断器
- 押しボタン
- 切替スイッチ
- リミットスイッチ
- ソレノイド弁
- センサ
- 電源
- トランス
- 接地
- モータ
- ファン
- 抵抗
- コンデンサ
- ダイオード
- サーマルリレー
- ケーブルマーカー
- ケーブル束
- 端子箱

部品パレットは、部品名または種別で検索/絞り込みできること。

表は等幅グリッドだけでなく、参照PDFの目次表・仕様表のような非等幅列/非等高行を再現できるよう、列幅、行高、表文字サイズを要素ごとに指定できること。

各要素は選択、移動、削除、複製、属性編集ができ、幅/高さを持つ単一要素は角ハンドルでリサイズできること。

a接点、b接点、押しボタンは、参照PDFの横配線上に見られる小型記号を既定形状とし、右パネルで「横向き標準」と「縦配線上の回転配置用」を切り替えられること。

接点ブロックは、`scan-001 (1).pdf p7A` の小型点線枠付き接点に合わせ、既定で横配線上へ中央配置でき、内部接点を a 接点 / b 接点から選択できること。

ヒューズ、遮断器、コイル、ランプ、ブザー、接地、モータ、抵抗、ダイオードは、参照PDF上の部品密度に合わせた小型寸法を既定とし、配線上に置きやすいインライン記号としてSVG表示とDXF出力の両方で同じ形状になること。

ヒューズと遮断器は、参照PDFの横配線上/縦配線上の両方の表現に対応し、縦配線上へ配置した場合は自動で縦向き寸法へ切り替わり、上下端子位置で既存配線を分割できること。

ランプは丸形パイロットと筒形表示灯を切り替えられ、ブザーは参照PDFのホーン形状とスピーカー形状を切り替えられること。

主要電気シンボルは、パレットから新規配置した時点およびテンプレート/自動生成で追加された時点で、参照PDF基準の既定寸法になっていること。参照寸法にするための後操作を前提にしないこと。

単体端子は、参照PDFの丸端子表記に合わせた小型の丸端子を既定とし、右パネルで丸/角を切り替えられること。

複数選択した要素は、Shiftクリックまたは空白ドラッグの範囲選択で選択でき、まとめて移動、コピー、複製、削除、重なり順変更、左/中央/右/上/下揃え、横/縦均等配置ができること。

機器箱は、参照PDFの小型機器箱に合わせ、左右の引出端子と左右別の端子文字を保持し、SVG表示とDXF出力に同じ位置で出力できること。

配線端点は、グリッドだけでなく端子、端子台ノード、接点端、コイル端、ランプ端、ブザー端、機器端子、接続点、既存配線端点へ吸着すること。接続済みの配線は、接続先シンボルを移動したときに端点が追従し、離れないこと。

接点、コイル、ヒューズ、遮断器、ランプ、ブザーなどのインライン部品を既存配線上へ配置した場合、配線が部品内部を貫通しないよう、部品の左右端子位置で既存配線を自動分割すること。

選択、配線、文字、直線、矩形、円などの主要作図ツールはキーボードで切り替えできること。

単一要素または複数要素の回路をユーザー部品として登録し、パレットから再配置できること。

## 7. 電気図面専用要件

- 配線には線番を付与できる。
- 線番は手入力と自動付与の両方に対応する。
- 線番書式は `%S-%N` や `%I-%L-%N` のような置換形式で設定できる。
- 配線種別を選択できる。
- 配線種別は名称、色、線幅、破線、自動線番対象外を編集・追加できる。
- 配線ごとに線番固定と線番非表示を設定できる。
- 線番固定された配線は自動線番で上書きしない。
- 配線ごとに中央ギャップと交差ギャップを追加・解除でき、交差線や参照線の表現に使用できる。
- 配線へ中間点を追加・削除し、折れ曲がり配線を点列として編集でき、端点/中間点は図面上のハンドルでも移動できる。
- 斜め区間を含む配線は、折れ点を追加して水平/垂直の直交配線へ整形できる。
- 本数、開始X、上下Y、ピッチ、線番接頭辞を指定して縦配線列を一括生成できる。
- 本数、左右X、開始Y、ピッチ、線番接頭辞を指定して横配線行を一括生成できる。
- 配線の FROM/TO は自動推定と手動指定の両方に対応し、配線順序ダイアログで一覧編集できる。
- 既存端点・端子・接続点への端点スナップに対応する。
- 配線端点や直交交差から接続点を自動生成できる。
- 端子台とコネクタは端子数、開始番号、方向、端子形状、接頭辞を編集できる。
- 端子番号をまとめて振り直せる。
- 端子台エディタで複数の端子台/コネクタを一覧編集できる。
- 端子 CSV/TSV を読み込み、端子番号、線番、信号名、接続先から端子接続図ページを自動生成できる。
- 端子 CSV/TSV は列名あり、列名なしのどちらにも対応し、行数が多い場合は自動でページ分割する。
- PLC I/O CSV/TSV を読み込み、アドレス、入出力方向、信号名、線番、接続先から PLC I/O 接続図ページを自動生成できる。
- 部品タグを自動採番できる。
- 部品タグ書式は `%F%N`、`%F-%I-%L-%N` などの置換形式で設定できる。
- 部品タグ書式は `%F`、`%N`、`%S`、`%P`、`%D`、`%I`、`%L` の置換に対応する。
- `%N` を含まない部品タグ書式でも、連番が失われないよう末尾に `%N` を補完する。
- 部品種別ごとのファミリ接頭辞を編集できる。
- 部品タグ自動採番の開始番号を指定できる。
- 部品タグ固定を設定した部品は自動採番で上書きしない。
- 信号矢印には参照、発信元、受信先の種別を設定できる。
- 線番、信号矢印、部品タグのクロスリファレンスを確認できる。
- 発信元/受信先の信号矢印が片側だけになっている場合は監査で検出できる。
- 部品・配線・端子の情報を帳票 CSV として出力できる。
- 帳票 CSV には配線の FROM/TO 推定情報を含める。
- 部品・配線・端子の帳票を、現在ページへ表部品として挿入できる。
- 帳票表挿入では、設備属性または場所属性で表示行を絞り込める。
- 図面監査として、未採番配線、未採番部品タグ、空ラベル、用紙外要素、未接続端点、信号矢印の片参照、端子台名称重複、同一部品種別のタグ重複などを検出できる。
- 図面監査結果から、対象要素を選択して該当ページへ移動できる。
- 各要素には自由記入の属性メモを保持でき、検索と帳票に反映する。
- 主要な電気部品には、タグ、説明、設備、場所、メーカー、品番、定格、項目の属性を保持できる。
- 電気属性は検索対象に含め、帳票 CSV では独立した列として出力する。
- 電気属性は項目ごとに図面上の表示/非表示を切り替えられ、表示位置と文字サイズを調整できる。
- 設備枠/場所枠を配置し、枠内の電気部品へ設備または場所属性を一括適用できる。
- 選択した電気部品の設備/場所属性をコピーし、別の電気部品または複数選択へ貼り付けできる。
- 図面内の線番、タグ、文字、属性メモを検索し、結果から該当要素を選択できる。

## 8. 保存・出力要件

- JSON 保存に対応する。
- JSON 読込に対応する。
- PDF 印刷に対応する。
- 現在ページの SVG 出力に対応する。
- 現在ページの PNG 出力に対応する。
- 現在ページの DXF 出力に対応し、配線ギャップは線分分割として反映する。
- 帳票 CSV 出力に対応する。
- ブラウザ内の自動バックアップから復元できる。

## 8.1 ドキュメント要件

- 全機能の実装後、初心者にもわかる詳細な取扱説明書を Markdown 形式で作成する。
- 取扱説明書は、起動方法、画面構成、ページ作成、図枠選択、部品配置、属性編集、線番、端子番号、保存、読込、印刷、各種出力、監査、トラブル対応を網羅する。
- Markdown 版の取扱説明書を PDF 化する。
- PDF 化した取扱説明書は `docs/PDF/` 内に保存する。
- 取扱説明書は `docs/manual.md` と `docs/PDF/manual.pdf` を成果物とする。

## 9. AutoCAD Electrical 踏襲方針

Autodesk 公式ヘルプに記載されている AutoCAD Electrical の考え方を参考に、以下の領域を段階的に実装する。

- プロジェクト/ページ管理
- 回路図作成
- 配線と線番
- 端子と端子台
- PLC / I-O
- 図枠/表題欄
- 帳票
- 図面監査
- 後工程での盤レイアウト、クロスリファレンス、シンボルビルダー相当機能

ただし、Autodesk 製品の画面、文言、マニュアル本文をそのまま複製するのではなく、業務フローと機能概念を参考にした独自実装とする。

- 帳票出力形式

## 11. 初期受入条件

- 単一 HTML ファイルをブラウザで開ける。
- 初期状態で A4 縦の図枠付き図面が表示される。
- 図枠パターンを切り替えられる。
- A4/A3、縦/横を切り替えられる。
- ページタブで複数ページを切り替えられる。
- 左パレットから部品を配置できる。
- 配線、端子台、接点、コイル、ランプ、ブザー、表を配置できる。
- 属性を編集できる。
- JSON 保存/読込ができる。
- PDF 印刷できる。
- 参照PDFの主要な図面要素を手作図で再現できる。

## 12. 今後の実装優先度

1. 参照PDFの各ページを再現するための不足記号を追加する。
2. 図枠テンプレートの精度を参照PDFに近づける。
3. 線番、端子番号、帳票、監査を強化する。
4. シンボルライブラリを外部定義化し、標準パック方式にする。
5. クロスリファレンス、PLC I/O 表、端子台エディタ相当機能を追加する。
6. トヨタ標準仕様が確定した時点で、専用テンプレートと検査ルールを追加する。
