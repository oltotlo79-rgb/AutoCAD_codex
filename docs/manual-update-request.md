**追加する内容**（下記をベースに、manual.md の文体・トーンに合わせて調整可）:

```markdown
### 5.2b ページ表示モード（タブ／並べて表示）

左パネル「表示」の **「ページ表示」** で、ページの表示方法を切り替えられます。

- **タブ（1ページずつ）**: 既定。上部タブで切り替えながら1ページずつ表示・編集します。
- **並べて表示（横）**: 全ページを**横一列**に並べ、どのページも直接編集できます。
- **並べて表示（縦）**: 全ページを**縦一列**に並べます。

並べて表示では:

- **クリックしたページが自動的に編集対象**（アクティブ）になります。
  アクティブページは**緑の枠と緑の見出し**で強調され、上部タブの選択も連動します。
- 部品の配置・配線・ダブルクリック編集・右クリックメニューなど、
  すべての操作を**どのページに対しても直接**行えます。
- 各ページ見出しの **◀▶（縦のときは▲▼）** ボタンで、
  ページの**並び順をその場で入れ替え**られます（タブ順・印刷順も連動します）。
- 上部タブをクリックすると、そのページへスクロールして編集対象になります。

ページをまたぐ比較・転記・全体の見通し確認に便利です。
ページ数が多い図面（目安10ページ超）では表示が重くなるため、
通常の作図はタブ表示、確認時に並べて表示、という使い分けをおすすめします。
```

## 動作仕様の正確な参照（執筆時の裏取り用）

- 設定値: `state.settings.multiPageView`（boolean）と `state.settings.multiPageDirection`（"row"＝横・既定 / "column"＝縦）。図面JSONに保存されます。
- 実装関数: `renderCanvasView()` / `activatePage()` / `movePageBySlot()`（index.html内、「ページ表示モード」コメントブロック）。
- 並べて表示中のドラッグ操作は編集中ページのみ再描画する軽量モードで動作します。
- 印刷・PDF・SVG/PNG/DXF出力は表示モードの影響を受けません（従来どおり）。
