# 取扱説明書 更新依頼（2026-07-23 の機能追加・修正分）

docs/manual.md へ反映してほしい変更の一覧です。各項目に「動作仕様」と「マニュアルへ書くべき内容」を記載します。
記載箇所の指定がないものは、関連する既存章へ追記してください。

## 1. 右パネル入力の確定仕様の変更（意図しない上書きの修正）

**動作仕様:**
- 右パネルの入力欄（ラベル・タグ・数値など）は、**パネルを開いた時点の要素へ束縛**される。
- 入力後に Enter を押さずに別のシンボルをクリックしても、入力値は**編集していた要素に確定**され、
  クリックした先のシンボルへ書き込まれることはない（従来はクリック先のラベルが書き換わる不具合があった）。

**マニュアルへ:**
- 「選択と編集」系の章に、Enter不要で確定される旨と、別要素クリックで値が失われたり誤適用されたりしない旨を1〜2文で追記。

## 2. 端子箱（terminalBox）の機能拡張

**動作仕様:**
- 右パネルに専用の設定が追加された:
  - **箱の数**（`rows`、最小1。従来は最小2で変更UIなし）
  - **箱ごとの引き出し線**: 左右それぞれ・箱ごとにチェックボックスでON/OFF（既定は全てON）。
    内部プロパティは `terminalBoxStubs`（例: `left-0`/`right-2` のキーに boolean）。
- 引き出し線の長さは左右とも 6mm（DXF出力も同じ長さ・同じ表示判定に統一。従来DXFは5mmだった）。
- **配線接続点の位置**:
  - 引き出し線がある箱 → **線の先端**（箱の辺から6mm外側）が主接続点。箱の辺中央も接続点として残る
    （既存図面で辺へ直接つないだ配線を未接続扱いにしないため）。
  - 引き出し線がない箱 → 箱の**横辺中央**のみ。
- **拡大縮小時のずれ修正**: 端子行の中央位置の2.5mm丸めを廃止し、箱の区画中央へ正確に配置。
  リサイズしても区画線・引き出し線・接続点が一致し続ける。

**マニュアルへ:**
- 端子箱の説明箇所（部品説明の章）に、箱の数・引き出し線の個別設定・接続点の位置ルールを追記。
- スクリーンショット更新推奨（引き出し線を一部OFFにした例があると分かりやすい）。

## 3. クリック当たり判定の改善

**動作仕様:**
- **斜めに配置した直線**（および機械連動線・信号矢印）の選択当たり判定を、従来の直交L字経路から
  **実際に描画される直線に沿った判定**へ修正。斜め線が再選択できない問題が解消。
- **寸法**は測定線に加えて**引出線・寸法線（オフセット位置）**もクリックで選択可能に。
- **接続点（黒丸）**は中心対称の円で判定（従来は右下寄りの矩形でクリック位置によって選べなかった）。
- 当たり判定の線幅を2.4mm→3mm相当へ拡大。

**マニュアルへ:**
- 「困ったときは（逆引き）」に「斜めの線が選択しづらい→線のすぐ上をクリック、Alt+クリックで重なり順選択」を追記済みのヘルプと同等の記述を追加。

## 4. ページ設定「シンボル拡大」チェックボックス（新機能）

**動作仕様:**
- 右パネル「ページ設定」に**「シンボル拡大（新規配置を1.5倍）」**チェックボックスを追加（ページごとの設定、保存対象）。
- ONの間にパレットから配置したシンボルは**1.5倍**で配置される（内部プロパティ `scale: 1.5`）。
  - 既存要素・配線/直線などの点列要素・テンプレート/回路ライブラリの一括生成は対象外。
  - 接続ピン位置・配線吸着・DXF出力（線・円・弧・文字サイズ）・選択ハンドル・リサイズはすべて倍率に追従し、デザインは崩れない。
  - 標準シンボル（ピンピッチ10mm）は1.5倍で15mmピッチ（=2.5mmグリッド整合を維持）。
- OFFに戻すと以後の配置は等倍。配置済みの1.5倍シンボルはそのまま残る。

**マニュアルへ:**
- 「ページ設定」章に項目追加。倍率は新規配置のみに効く点、既存要素へ影響しない点を明記。

## 5. 選択時ミニツールバーの表示位置変更

**動作仕様:**
- シンボル選択時に出る「配線/回転/複製/編集」（配線は「線番/曲げ/分割/削除」等）のミニツールバーを、
  要素から**さらに離れた上方**（収まらない場合は**要素の下方**）へ表示するよう変更。
  選択ハンドルや接続点の操作と重ならない。

**マニュアルへ:**
- ミニツールバーの説明があれば位置の記述を更新。なければ「選択と編集」章にボタンの機能一覧とあわせて追記。

## 6. ヘルプ（使い方ガイド）の拡充

**動作仕様（参考。マニュアル本文への影響は小）:**
- ヘルプに以下を追記済み: シンボル拡大 / 端子箱の箱数・引き出し線 / ミニツールバー / 入力確定仕様 /
  斜め線の選択のコツ（Alt+クリック含む）。
- マニュアルとヘルプの記述が食い違わないよう、上記1〜5の反映時にヘルプ文面（index.html内 showHelpDialog）も参照のこと。
