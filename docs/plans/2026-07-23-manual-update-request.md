# 取扱説明書 更新依頼（2026-07-23 の機能追加・修正分）

docs/manual.md へ反映してほしい変更の一覧です。各項目に「動作仕様」と「マニュアルへ書くべき内容」を記載します。
記載箇所の指定がないものは、関連する既存章へ追記してください。

## 1. 右パネル入力の確定仕様の変更（意図しない上書きの修正）

**動作仕様:**
- 右パネルの入力欄（ラベル・タグ・数値など）は、**パネルを開いた時点の要素へ束縛**される。
- 入力後に Enter を押さずに別のシンボルをクリックしても、入力値は**編集していた要素に確定**され、
  クリックした先のシンボルへ書き込まれることはない（従来はクリック先のラベルが書き換わる不具合があった）。

**マニュアルへ:**
- 「選択と編集」系の章に、Enter不要で確定される旨と、別要素クリックで値が失われたり誤適用されたりしない旨を1〜2文で追記。

## 2. 端子箱（terminalBox）の機能拡張

**動作仕様:**
- 右パネルに専用の設定が追加された:
  - **箱の数**（`rows`、最小1。従来は最小2で変更UIなし）
  - **箱ごとの引き出し線**: 左右それぞれ・箱ごとにチェックボックスでON/OFF（既定は全てON）。
    内部プロパティは `terminalBoxStubs`（例: `left-0`/`right-2` のキーに boolean）。
- 引き出し線の長さは左右とも 6mm（DXF出力も同じ長さ・同じ表示判定に統一。従来DXFは5mmだった）。
- **配線接続点の位置**:
  - 引き出し線がある箱 → **線の先端**（箱の辺から6mm外側）が主接続点。箱の辺中央も接続点として残る
    （既存図面で辺へ直接つないだ配線を未接続扱いにしないため）。
  - 引き出し線がない箱 → 箱の**横辺中央**のみ。
- **拡大縮小時のずれ修正**: 端子行の中央位置の2.5mm丸めを廃止し、箱の区画中央へ正確に配置。
  リサイズしても区画線・引き出し線・接続点が一致し続ける。

**マニュアルへ:**
- 端子箱の説明箇所（部品説明の章）に、箱の数・引き出し線の個別設定・接続点の位置ルールを追記。
- スクリーンショット更新推奨（引き出し線を一部OFFにした例があると分かりやすい）。

## 3. クリック当たり判定の改善

**動作仕様:**
- **斜めに配置した直線**（および機械連動線・信号矢印）の選択当たり判定を、従来の直交L字経路から
  **実際に描画される直線に沿った判定**へ修正。斜め線が再選択できない問題が解消。
- **寸法**は測定線に加えて**引出線・寸法線（オフセット位置）**もクリックで選択可能に。
- **接続点（黒丸）**は中心対称の円で判定（従来は右下寄りの矩形でクリック位置によって選べなかった）。
- 当たり判定の線幅を2.4mm→3mm相当へ拡大。

**マニュアルへ:**
- 「困ったときは（逆引き）」に「斜めの線が選択しづらい→線のすぐ上をクリック、Alt+クリックで重なり順選択」を追記済みのヘルプと同等の記述を追加。

## 4. ページ設定「シンボル拡大」チェックボックス（新機能）

**動作仕様:**
- 右パネル「ページ設定」に**「シンボル拡大（新規配置を1.5倍）」**チェックボックスを追加（ページごとの設定、保存対象）。
- ONの間にパレットから配置したシンボルは**1.5倍**で配置される（内部プロパティ `scale: 1.5`）。
  - 既存要素・配線/直線などの点列要素・テンプレート/回路ライブラリの一括生成は対象外。
  - 接続ピン位置・配線吸着・DXF出力（線・円・弧・文字サイズ）・選択ハンドル・リサイズはすべて倍率に追従し、デザインは崩れない。
  - 標準シンボル（ピンピッチ10mm）は1.5倍で15mmピッチ（=2.5mmグリッド整合を維持）。
- OFFに戻すと以後の配置は等倍。配置済みの1.5倍シンボルはそのまま残る。

**マニュアルへ:**
- 「ページ設定」章に項目追加。倍率は新規配置のみに効く点、既存要素へ影響しない点を明記。

## 5. 選択時ミニツールバーの表示位置変更

**動作仕様:**
- シンボル選択時に出る「配線/回転/複製/編集」（配線は「線番/曲げ/分割/削除」等）のミニツールバーを、
  要素から**さらに離れた上方**（収まらない場合は**要素の下方**）へ表示するよう変更。
  選択ハンドルや接続点の操作と重ならない。

**マニュアルへ:**
- ミニツールバーの説明があれば位置の記述を更新。なければ「選択と編集」章にボタンの機能一覧とあわせて追記。

## 6. ヘルプ（使い方ガイド）の拡充

**動作仕様（参考。マニュアル本文への影響は小）:**
- ヘルプに以下を追記済み: シンボル拡大 / 端子箱の箱数・引き出し線 / ミニツールバー / 入力確定仕様 /
  斜め線の選択のコツ（Alt+クリック含む）。
- マニュアルとヘルプの記述が食い違わないよう、上記1〜5の反映時にヘルプ文面（index.html内 showHelpDialog）も参照のこと。

## 7. 文字サイズ入力の増減幅を0.1mmへ統一

**動作仕様:**
- 右パネルの文字サイズ系の数値入力（文字要素の文字サイズ／部品・配線のラベル文字サイズ／
  属性メモの表示文字サイズ／文字スタイルの文字サイズ／表文字サイズ）の上下ボタン（スピナー）を
  すべて**0.1mm刻み**に統一（従来は項目により0.2mm刻みが混在していた）。

**マニュアルへ:**
- 文字サイズの説明箇所に刻み幅の記述があれば0.1mmへ修正。なければ変更不要（挙動の統一のみ）。

## 8. 端子箱の箱内文字（新機能）

**動作仕様:**
- 右パネルの端子箱設定に**「箱内文字（1行=1箱）」**のテキスト欄と**「箱内文字サイズ mm」**（既定2.2、0.1刻み）を追加。
- 入力した各行が対応する箱の**区画中央**に表示される（内部プロパティ `boxLabels` / `boxLabelFontSize`）。
- DXF出力にも同じ位置・サイズで文字が出る。

**マニュアルへ:**
- 端子箱の説明箇所（§2の箱数・引き出し線と同じ場所）へ追記。相番号（R/S/T等）の記入例があると親切。

## 9. 選択クリックだけでは要素が吸着で動かないように修正

**動作仕様:**
- 従来、正確な位置に合わせた**直線・連動破線・矢印など**をクリックして選択しただけで、
  端点が近く（4mm以内）のグリッド・配線接続点へ吸着し、位置がずれてしまうことがあった。
- 修正後は、**選択クリックのみでは吸着しない**。吸着が働くのは
  **要素をドラッグで移動したとき**と**端点ハンドルをドラッグしたとき**（および作図の確定時）のみ。

**マニュアルへ:**
- 吸着（オブジェクトスナップ）の説明箇所に「選択だけでは動かない／移動・端点ドラッグ・作図時のみ吸着」の1文を追記。

## 10. 矩形・多角形の配線接続点（新機能）

**動作仕様:**
- **矩形**: 4頂点+4辺の中央、**多角形**: 全頂点+各辺の中央（閉じる辺を含む）が配線の吸着先になった。
- 配線ドラッグ・端点ドラッグの吸着対象になり、選択中の「配線」ボタン（ミニツールバー/右パネル）からも
  これらの接続点を開始点に選べる。
- 用途: 囲み枠・機器外形の絵へ引き込み線や指示線を正確に接続する。

**マニュアルへ:**
- 配線の章の吸着先一覧（端子・接点ピン等）へ「矩形・多角形の頂点/辺中央」を追加。

## 11. コンセントのデザイン拡充と「箱形（テンプレート専用10mm）」の廃止

**動作仕様:**
- コンセントのシンボルデザインを6種へ拡充:
  - `standard` 半円＋差込口（従来どおり）
  - `box` 箱形（スロット2＋接地）
  - `boxDouble` **箱形2口**（差込口スロット2組＋接地バー）〔新規〕
  - `round` **丸形（円＋差込口2本）**〔新規〕
  - `earthed` **半円＋差込口（接地極付き＝接地記号併記）**〔新規〕
  - `plugSocket` **プラグ・ソケット対（挿抜表現、2ピン10mmピッチ）**〔新規〕
- **「箱形（テンプレート専用10mm）」（`boxGrid10`）は廃止**:
  - `box` の受電2ピンを9.5mm→**10mmピッチ**（ローカル(0,2)/(0,12)）へ統一し、boxGrid10と同一形状にした。
  - 旧図面は読込時の互換補修で自動移行: `boxGrid10`は`box`へ改名（形状同一のため配線は動かない）、
    旧9.5mmピッチの`box`は第2ピン位置(0,11.5)に接続していた配線・接続点だけが(0,12)へ追従する。
  - ACテンプレートの盤内コンセント(SC1/SC2)も`box`を使用するようになった。

**マニュアルへ:**
- コンセントのデザイン一覧表を6種へ更新し、「テンプレート専用10mm」の記述を削除。
- 旧図面の自動移行（ピン0.5mm移動と配線追従）について1文注記。

## 12. ラベル文字サイズ入力の一本化

**動作仕様:**
- 従来、ラベルを持つシンボルでは「ラベル文字サイズ」と「文字スタイル＞文字サイズ」の**2つの入力が重複**していた
  （どちらもラベルの文字サイズを変更し、後者が優先される）。
- **「文字スタイル」の文字サイズへ一本化**し、ラベル欄からサイズ入力を削除（ラベル位置X/Yは残る）。
  直線の「線ラベル」も同様に文字スタイル側で変更する。
- 旧図面で`labelFontSize`が設定済みの場合は従来どおり描画に反映される（互換維持）。
  文字スタイルの初期表示値も旧設定を引き継ぐ。

**マニュアルへ:**
- ラベル文字サイズの説明を「文字スタイル」章へ集約。旧記述（ラベル文字サイズ欄）があれば削除。

## 13. コンセントの雄側（プラグ）デザイン追加（計9種へ）

**動作仕様:**
- コンセントに雄側プラグ3種を追加（§11の6種と合わせて計9種）:
  - `plug` **プラグ（雄側・縦バー）**: リード＋縦バー（JIS C 0617系、ソケット対の雄半分）
  - `plugPin` **プラグ（2本刃）**: リード＋胴バー＋平行刃2本の絵記号
  - `plugPin3` **プラグ（2本刃＋接地ピン）**: 刃2本の中間に短い接地ピンを持つ接地2P形
- いずれも接続点は左1ピン。

**マニュアルへ:**
- コンセントのデザイン一覧表を9種へ更新（§11の表と統合して記載）。

## 14. 機器箱の端子文字サイズを独立指定

**動作仕様:**
- 機器箱の右パネルに**「端子文字サイズ mm」**（既定2.2、0.1刻み）を追加（内部プロパティ `pinFontSize`）。
- 端子文字（左右のピン名）は専用サイズで描画され、「文字スタイル」の一括文字サイズの影響を**受けない**。
- ラベル（箱中央の文字）は従来どおり「文字スタイル」の文字サイズで変更する。
- DXF出力も端子文字は`pinFontSize`で出力される。

**マニュアルへ:**
- 機器箱の説明に「ラベル=文字スタイル／端子文字=端子文字サイズ」の役割分担を追記。

## 15. 端子台・PLCユニット列の端子記号（新機能）

**動作仕様:**
- 端子台（terminalStrip）とPLCユニット列（plcBlockのユニット形式）の各端子行に、
  **機器箱・単体端子と同じ端子円（半径0.55mm・白抜き）を本体の辺上**へ表示できる
  （浮かせず辺に載せ、円内の線を白抜きで隠す＝機器箱の端子円と同じ見せ方）。
  端子円からは**盤外方向へ長さ5mmの破線スタブ**（パターン3-2、盤外/参考線と同じ。
  先端が配線接続点になるため2.5mmグリッドに整合する長さ）が出る。
  横向きピル形端子台は上下の辺に付く。SVG・DXFとも出力される（DXFの破線はDASHED線種）。
- 既定値:
  - **パレットから端子台を追加** → **左側のみON**
  - **PLC入出力回路テンプレート** → **入力回路=ユニット列の左側 / 出力回路=右側**にON
  - 端子接続図・端子CSV取込のページ、既存図面 → OFF（従来の見た目を維持）
- 右パネルの「端子記号」で**左右の一括トグル**（内部 `terminalMarkLeft`/`terminalMarkRight`）と、
  「端子ごとの表示」で**端子単位のON/OFF上書き**（内部 `stripTerminalMarks` の left-n/right-n）ができる。
- **端子記号は配線接続点になる**: 表示中の端子記号ごとに、**小円中心（本体の辺上）**と
  **破線スタブの先端（5mm外側）**の2点が配線の吸着先・配線開始点（選択中の「配線」ボタン、
  配線ツールの接続点マーカー）に追加される。従来の端子中心（端子円・アドレス円の中心）も引き続き有効。

**マニュアルへ:**
- 端子台の章とPLC入出力回路テンプレートの章の両方へ追記。テンプレート生成例のスクリーンショット更新推奨。

## 16. JIS網羅性の再監査で追加したシンボル3種

**動作仕様:**
- **フォトダイオード**（ダイオードのデザイン、値 `photo`）: LEDの対になる受光矢印（外→素子へ向く2本）。
  寸法・接続点は標準ダイオードと共通。
- **ポテンショメータ（3端子可変抵抗）**（抵抗形状、値 `potentiometer`）: 箱形抵抗+上からの矢印タップ。
  接続点は左右2ピンに加えて**タップ端子（上方向）**の3点。タップは左ピンから(+5, -5)mmで2.5mmグリッド整合。
  ラベルは矢印と重ならないよう箱の下側に表示。
- **DC/AC変換器（インバータ）**（電源のデザイン、値 `dcac`）: 箱+対角線、左上に⎓・右下に〜。
  寸法・接続点はAC/DC変換器と共通（w35×h25、左右2ピン）。

**マニュアルへ:**
- ダイオードのデザイン一覧へ `photo` を、抵抗形状一覧へ `potentiometer` を、電源一覧へ `dcac` を追加。

**網羅性の判断メモ（マニュアル記載は任意）:**
- JIS C 0617のうち展開接続図・制御盤図で常用する記号は、これまでの追加分（接触器主接点・トライアック・
  JISブザー・交流電源・サーマルリレー各種・プラグ/コンセント各種）と本節で網羅。
- 意図的に未収載: 建築屋内配線図記号（JIS C 0303の壁スイッチ等）、電子回路特化（FET・水晶振動子等）、
  確度の低い特殊開閉器（負荷開閉器・接地開閉器の細部意匠）。必要になれば見本図面を基に追加する方針。
