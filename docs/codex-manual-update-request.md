## 1. 非常停止ボタンのシンボル刷新（既定デザイン変更・重要）

- **既定シンボルが変更**: 従来の「キノコ＋ひねり解除環（設備標準）」から、一般的な **「キノコ＋b接点（一般形）」**（半円キノコ傘＋押込みで開くb接点）に変わった。
- 既存図面も既定（standard）のままの非常停止は新デザインで表示される。従来の見た目に戻すには、部品選択 → 右パネル「シンボルデザイン」→ **「キノコ＋ひねり解除環（設備標準）」** を選択（値: facility）。
- バリアントは5種類:
  | 値 | 表示名 | 説明 |
  |---|---|---|
  | standard | キノコ＋b接点（一般形） | 新しい既定。JIS/IEC系の一般的な非常停止表現 |
  | multi2 | 縦連結2接点（連動） | 1つのキノコ操作部で連動する2接点を縦に連結。破線=機械連動。接点間は行ピッチ9.5mmで隣接ラング2行に載る。接続点は各段の左右(計4) |
  | multi3 | 縦連結3接点（連動） | 同3接点版(接続点は計6) |
  | facility | キノコ＋ひねり解除環（設備標準） | 旧既定(見本PDF実測形) |
  | bar | バー押込式 | 従来どおり |
- 用途説明: 非常停止は複数接点が同時に連動して動作するものがあるため、縦連結バリアントで多接点を1シンボルとして作図できる。
- マニュアルの「シンボルデザイン一覧」表・非常停止の作図例・スクリーンショットの差し替えが必要。

## 3. テンプレート「AC電源回路（一括）」の全面刷新

見本PDF（AC電源_走行横行モータ / AC電源_昇降モータ）を忠実に再現する構成に変更。

- ダイアログ項目が変更: 相レール名(R01,S01,T01) / **接地レール名(G・空欄で省略)** / モータタグ(M1) / MC接点タグ(**MC2M1**) / サーマルタグ(OL1) / **遮断器タグ(CB1)** / **インバータタグ(INV1)** / **インバータ型式(FR-E720 0.4K)** / ブレーキ回路数(0-4) / **ブレーキ接点接頭辞(MSB)** / 通信ツイストペア有無 / 信号名(**DA,DB,DG,SLD**)
  - 旧「右レール名(E)」は廃止（見本に右レールは無い。接地Gレールが左端に付く）
- 生成内容:
  - 縦レール: G＋相3本（上下に送り矢印・上下端ラベル、G下端に⊕保安接地マーク）
  - ブレーキ回路×N: 相1/相3からのタップ縦線 → 接点2段(MSBn) → 整流器箱(ピン2/1入・4出) → ブレーキコイルBn（見本と同じループ配線、線番 B0n / Bn1 / Bn）

  - タップ線が逆方向に計算されS01/T01につながらない不具合
  - SIG1〜4等の信号名文字が縦レールに重なる不具合（信号名はチェブロン左に右寄せ配置へ変更）
  - レール位置が2.5mmグリッドに乗らない不具合（G=30 / 相=35,40,45mm）

## 4. 一括生成「非常停止チェーン」の刷新

見本PDF「リレー回路_非常停止」の様式に合わせて変更。

- 各局の右に境界端子(○)を追加し、局間の線番を自動付与（例: 3局チェーン0 → Si0B → Si0A → Si0。末尾区間が基準名、手前はA,B,…）

- 表示灯行が見本と同じ「同じPBの別接点 → 局内でループ配線 → Rランプ → N1」の形に変更（従来はa接点＋直線配線）

## 8. 非常停止ボタンに JIS C 0617 形と JIS 縦連結を追加（重要）

非常停止のバリアントが5種類→8種類になった。§1 の表へ以下の3行を追加:

| 値 | 表示名 | 説明 |
|---|---|---|
| jis | キノコ＋b接点（JIS C 0617） | JIS C 0617（IEC 60617）準拠の表現。キノコ操作部＋破線の機械連動＋IEC様式b接点（斜め可動線＋止め棒） |
| jisMulti2 | JIS縦連結2接点（連動） | JIS C 0617様式のb接点を縦に2段連結（破線=機械連動、段ピッチ9.5mm、接続点は各段左右の計4） |
| jisMulti3 | JIS縦連結3接点（連動） | 同3接点版（接続点は計6） |

- 既定（standard=一般形）は変わらない。寸法・接続点は一般形と共通のため切替で配線はずれない。
- 表示順は standard → jis → multi2 → multi3 → jisMulti2 → jisMulti3 → facility → bar。

## 10. テンプレート/一括生成の2.5mmグリッド整合（重要・全テンプレート）

「テンプレートで作成したページのシンボル・線がグリッドに乗らない」問題を修正した。設計方針（マニュアルの備考向け）:

- **X座標**: 配線端点・部品の左（基準）ピン・タップ縦線・箱の辺を2.5mmグリッドに乗せた。
  接点(幅8mm)・遮断器(幅12mm)・モータ(端子ピッチ5.6mm)など見本実測寸法の記号は、右側ピンがグリッドから外れることがある（見本再現を優先）。
- **Y座標**: 設備標準の行グリッド（27+9.5k mm）を維持（見本PDF準拠）。行に載らない自由なY（レール矢印の尻、枠上端など）はグリッドへ移動。
- **端子小円**: 中心をグリッド点に配置し、配線は円中心まで届かせて白抜き円で隠す方式に統一（従来は円の縁の手前 0.35mm で止まり微小な隙間があった）。

**全シンボルの接続ピン間隔を2.5mmグリッドの倍数に統一した。** 実測寸法由来でピッチが半端だった記号はリード線の延長などで調整（記号本体の見た目は維持）:

| 変圧器(バー/円形/鉄心) | 14mm・タップ(3.4, 0.9) | **15mm**・タップ(5/10, 0.5) | |
| モータ/発電機 | 端子ピッチ5.6mm・接地ピン(14.6,14.6) | **5mm**・接地ピン(13.4,15.5)=第1ピン+(12.5,12.5) | |
| ブザー・ファン | 幅14 | **15** | 既定寸法変更 |
| ソレノイド弁/センサ/電源/機器箱/端子箱 | 26/24/34/28×16/22×18 | **25/25/35/30×25/22.5×15** | 既定寸法変更 |

- **プロジェクトデータはschemaVersion 3へ移行**（読込時に自動マイグレーション）: 既存要素の寸法を新ジオメトリへ合わせ、旧ピン位置で終端していた配線・接続点(junction)を新ピン位置へ自動追従させる。旧図面の配線は切れない（検証済み）。

### 18-1. AC電源回路テンプレートの全面刷新（見本.pdf 頁4・頁12）

- **電源電圧を選択可能**: AC200V三相（G+R/S/T）／**AC100V単相**（G+R/N、変圧器なしで直接分岐）
- 電源表示灯PL0 → 漏電遮断器ELB0（縦3極アーチ形＋連動破線＋メインアースブロック点線枠＋端子）→ 送りレール（R01/S01/T01）
- CP分岐（すべてアーチ形サーキットプロテクタ2極＋連動破線）:
  盤内照明（CP0→切/入スイッチSS0→蛍光灯FL1）／ファン（CP01→FAN1）／REC送り（CP1→205/206矢印）／
  変圧器Tr1（**100VA/500VA/なし**を選択、200V/0V→100V/0V）→CP3→盤内コンセントSC1（箱形）／
  操作盤コンセント（CP4→CP11→SC2）／GOT電源送り（CP12→109/110矢印）
- 電線サイズ表（線番/サイズ/色）と SCCR注記を自動配置
- 旧インバータ構成の入力欄（モータタグ等）は「AC電源回路（モータ）」ダイアログへ移動

### 18-2. 新テンプレート「AC電源回路（モータ）」（見本.pdf 頁5）

駆動方式を選択:
- **サーボアンプ（2軸/1軸）**: CB（アーチ形3極）→MC接点→サーボアンプ箱（MR-J4W2形: L1-L3/L11/L12/P3-P4短絡/DICOM/EM2/DOCOM/MBR-A/B/CN1A）→U/V/W/PE→端子→サーボモータSM1/SM2（PEループ付き）＋CN2A/Bエンコーダ送り矢印＋ブレーキ回路（P24/N24受け→CRMBR接点→バリスタ＋ブレーキコイルB1/B2）＋N1/P3M受け・CRMBRリレーコイル（LY2N）
- **インバータ**: 従来の「AC電源回路（一括）」と同じ構成（遮断器→インバータ→MC→サーマル→モータ＋ブレーキ/整流＋CC-Linkツイストペア）

### 18-3. 新テンプレート「DC電源回路」（見本.pdf 頁6）

CP1（2極）→スイッチング電源REC1箱（L/N入力・-V/+V出力・接地）→N1（制御電源）/P0送り矢印、CP8→P24/N24（サーボモーターブレーキ電源）分岐、G送り行（⊕付き）。

### 18-4. 新テンプレート「PLC電源回路」（見本.pdf 頁7）

CP4→P0A→PLCベース箱（ベース名・スロット構成をカンマ区切りで指定、区切り線と型式を自動描画、DC+/DC-/⏚ピン）、N1→入出力COM／CP5→P1制御電源／CP6+接点2連→P3運転準備／CP7+接点2連→P3M動力供給の送り行。

### 18-5. 新テンプレート「セーフティリレー回路」（見本.pdf 頁8）

セーフティリレーユニット箱（PCR2E形）＋非常停止PB2個（**縦連結2接点**を使用）のチャンネル1/2直列ループ（T11-T12/T21-T22）、B-A短絡（オートリセットモード）、出力接点13-14/23-24（1N51→1N52等の送受矢印）、41-42→非常停止モニタリレーCR11、A1-A2ユニット電源行。
